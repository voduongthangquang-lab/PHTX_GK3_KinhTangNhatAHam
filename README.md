# Quiz App Documentation

## Overview
This Quiz App is designed to provide users with a seamless experience for taking quizzes on various topics. It incorporates features such as timed questions, multiple-choice options, instant feedback, and a scoring system.

## Features
- **User-Friendly Interface**: Simple and intuitive design for ease of use.
- **Timed Quizzes**: Users can take quizzes within a specified time limit.
- **Multiple Choice Questions**: Users can select answers from multiple options.
- **Instant Feedback**: Provides immediate feedback after each question.
- **Score Tracking**: Keep track of the user’s score and progress.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/voduongthangquang-lab/PHTX_GK3_KinhTangNhatAHam.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PHTX_GK3_KinhTangNhatAHam
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the app, run the following command:
```bash
npm start
```
